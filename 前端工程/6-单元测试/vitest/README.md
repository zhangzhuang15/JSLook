## Get Started
### run unit test
```sh 
$ pnpm unit-test
```
#### run benchmark test 
```sh 
$ pnpm bench-test
```
#### run coverage test 
```sh 
$ pnpm coverage
```

#### how to create this demo 
1. `pnpm install -D vitest`
2. write your `vitest.config.ts` following vitest official intro
3. write your source code, try to test them

there's no scaffold like `pnpm create vite`
